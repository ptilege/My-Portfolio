About Me...

I am a dedicated and detail-oriented Quality Assurance (QA) Engineer with experience in both manual and automation testing. Currently working at Adeona Technologies, I have gained extensive hands-on experience in delivering high-quality software solutions by performing thorough testing across various platforms.

My expertise includes:



  • Manual Testing: Conducting functional, regression, and exploratory testing to ensure software meets business requirements and end-user needs.

  • Automation Testing: Leveraging tools such as Selenium and Rest Assured to automate web and API tests, increasing testing efficiency and coverage.

  • API Testing: Using Postman and Rest Assured for comprehensive API testing, ensuring seamless communication between system components.

  • Performance and Load Testing: Utilizing JMeter to simulate real-world traffic conditions and assess system performance under varying loads.

  • Cross-browser and Cross-device Testing: Ensuring applications are responsive and work flawlessly across various browsers and devices.

  • Continuous Integration: Integrating test automation with CI/CD pipelines to deliver quicker, more reliable software releases.



I am passionate about ensuring the highest software quality through rigorous testing methods and constant learning. My approach combines solid technical skills with strong communication and problem-solving abilities to deliver results that exceed expectations.
